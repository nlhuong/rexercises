NASA’s online data portal includes many interesting datasets you can download. One of them is a datasets listing all the NASA facilities available at http://open.nasa.gov/datasets/NASA_Labs_Facilities.xlsx.

Download the ‘NASA_Labs_Facilities.xlsx’ file to your ‘Downloads’ directory:

url1 <- "http://open.nasa.gov/datasets/NASA_Labs_Facilities.xlsx"

# Download to e.g. the Download directory
download.file(url1, destfile = "./NASA_Facilities.xlsx")

Note that R generated some column names e.g. “X__1“, this is because the Excel file we used included some merged columns. If you’d like to remove the top merged column names, use the ‘skip’ argument when calling ‘read_excel()’ function:

nasa_fc <- read_excel("NASA_Facilities.xlsx", skip = 1)
nasa_fc.df <- as.data.frame(nasa_fc)
head(nasa_fc.df)
nasa_fc <- readxl::read_excel("NASA_Facilities.xlsx", skip = 1)
nasa_fc.df <- as.data.frame(nasa_fc)
