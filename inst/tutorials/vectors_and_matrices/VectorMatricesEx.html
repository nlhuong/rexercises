Exercise 1.2

Note that: In the previous section, the two calls to the same function with the same input and arguments give two different answers. This is because the sample() function generates random numbers.

Now repeat the same process, but set a random seed before generating again the two random vectors.

set.seed(123)
sample(1:100, size = 10)
set.seed(123)
sample(1:100, size = 10)

Note that: Now two vectors are the same; set.seed() function sets the random number generator. You can save the seed number and report it or give it to your collaborators, so that they can reproduce your results if any methods you used involved randomness.
