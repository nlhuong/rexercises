In this exercise we will plot the trends in the U.S. energy consumption by source. First, read the data from https://www.eia.gov/totalenergy/data/browser/csv.cfm?tbl=T01.03

energy <- read.csv("https://www.eia.gov/totalenergy/data/browser/csv.cfm?tbl=T01.03",
                    stringsAsFactors = FALSE)
energy$Value <- as.numeric(energy$Value) # convert to numeric values
knitr::kable(head(energy))

Preprocessing:

In the data.frame ‘energy’, the column ‘Description’ gives the type of enery source. Column ‘Value’ gives the actual consumption in Quadrillion BTU units.

Note that, the data.frame ‘energy’ contains a column ‘YYYYMM’ which stands for year and month of the measurement. We will convert this data to a date format in R:

energy$Year <- sapply(energy$YYYYMM, function(x) {substr(x, 1, 4)})
energy$Month <- sapply(energy$YYYYMM, function(x) {substr(x, 5, 6)})

There are some entries in the data that indicate month ‘13’, which does not make sense. Since, we do not know what exactly this means, we will treat these entries as invalid and remove them:

energy <- energy[energy$Month != "13", ]
knitr::kable(head(energy))

Use the plot() function to produce a scatter plot of the ‘Coal Consumption’ over time using column ‘YYYYMM’ and ‘Value’

# First subset the data:
coal <- energy[energy$Description == "Coal Consumption", ]
# Then use plot() on coal data.frame to plot Value over time
coal <- energy[energy$Description == "Coal Consumption", ]
plot(coal$YYYYMM, coal$Value)

Since monthly data might be too noisy, we will aggreagate the total annual consumption for each type of energy source:

# Check ?agreggate for function documentation
annual_energy <- aggregate(Value~Description+Year, energy, sum)
knitr::kable(head(annual_energy))

Also, we will remove the records for 2017, as not a whole year has past and it would not be thus, a fair comparison:

# Check ?agreggate for function documentation
annual_energy <- annual_energy[annual_energy$Year != 2017, ]

Now, extract the coal consumption from ‘annual_energy’ and then generate a line plot of the annual trands in its consumption:

coal <- annual_energy[annual_energy$Description == "Coal Consumption", ]
plot(x = coal$Year, y = coal$Value, type = "l")
